# RAG_BPMN
## Бизнес-/прикладная задача: 
Автоматизация проектирования основы для создания моделей бизнес-процессов.
### Способ решения: 
Разработка двухэтапного RAG-конвейера для генерации BPMN-схем и JSON-структур процессов на базе LLM.
## Предметная область: 
Корпоративное управление и моделирование процессов.
### Способ решения: 
Анализ текстовых регламентов экспертизы правовых актов для извлечения ролей, шлюзов и связей.
## Необходимость применения методов машинного обучения: 
Высокая вариативность речевых паттернов в текстах.
### Способ решения: 
Семантический анализ документов моделью Qwen 14B для точного распознавания неявной логики процессов.
## Тип задачи: 
Извлечение структурированной информации из текста.
### Способ решения: 
Перевод неструктурированного PDF-документа в валидный XML-код BPMN и типизированную JSON-таблицу.
## Ограничения: 
Аппаратные лимиты видеопамяти графического ускорителя Tesla T4.
### Способ решения: 
Жесткое кэширование контекста до 32000 токенов и ограничение длины ответа через параметры функции.
## Допущения: 
Эталонные схемы процессов в базе данных безошибочны.
### Способ решения: 
Извлечение чистых XML-аналогов из PostgreSQL для формирования контекста RAG на основе релевантных ID.
## Ожидаемый эффект от внедрения решения: 
Сокращение времени на отрисовку графических моделей.
### Способ решения: 
Мгновенное получение готовой архитектуры процесса для последующей автоматической визуализации на холсте Draw.io.
